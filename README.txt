Простой парсер для парсинга акционных товаров из https://vsem-darom.ru
Получаемые данные:
*"Наименование и арттикул",
*"Стоимость",
*"Наличие",
*"Изображение"
Полученные данные сохраняются в csv файл.
Установка:
1) Скачивайте репозиторий
2) Устанавливаете библиотеки: pip install request, BeautifulSoup4, lxml
3) Запускаете выполнение кода
